WooCommerce XRP is a payment gateway that let you accept XRP as payments.

== Description ==
WooCommerce XRP is a payment gateway which makes it easy to accept [XRP](https://ripple.com/xrp) as a payment method. It currenty uses [Bitstamp](https://www.bitstamp.net/) for rates and [XRPL Webhooks](https://webhook.xrpayments.co) to get notifications when payments has arrived.

== Installation ==
Download and unpack in wp-content/plugins
